# Video browser

Create a video browser with YouTube API using React and axios.

- [View Demo](https://uxmoon-video-browser.netlify.app/)

## Requirements

- NodeJs
- Google API Key

## Usage

- Run `yarn` to install dependencies
- Run `yarn start` to run app
